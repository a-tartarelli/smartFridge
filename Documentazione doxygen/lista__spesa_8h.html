◆ aggiungi_a_lista()

unsigned short int aggiungi_a_lista ( char  nome_file[],
char *  alimento 
)

Procedura che si occupa di aggiungere un elemento alla lista della spesa. Se l'alimento e' gia' presente in lista allora non verra' aggiunto.

Parametri
nome_fileindica il nome del file su cui verranno salvati i vari alimenti da aggiungere alla lista
alimentoindica l'alimento da aggiungere alla lista della spesa
Restituisce
1 se l'alimento viene aggiunto alla lista della spesa
0 se l'alimento non viene aggiunto alla lista della spesa
Precondizione
il file deve gia' esistere in memoria, altrimenti la lettura e la conseguente scrittura non verra' effettuata
