◆ confronta_date()

unsigned short int confronta_date ( data  d1,
data  d2 
)

La funzione confronta prima l'anno, poi il mese e poi il giorno. Se tutti i contronti danno come risultato dei valori uguali, le due date saranno uguali. Se invece una data e' minore dell'altra lo si evince dal confronto in sequenza di anno, mese e giorno. Lo stesso vale per verificare che una data e' maggiore dell'altra.

Precondizione
le date in input devono essere corrette e i campi devono essere pieni, altriemnti potrebbero verificarsi errori logoci a runtime
Postcondizione
l'intero restituito deve essere coerente con il confronto delle date (es. non deve restituire 0 se d1 > d2)

◆ giorno_precedente()

data giorno_precedente ( data  data_corrente)

La funzione esegue dei controlli sul mese e sul giorno in modo da evitare di avere date errate, come ad esempio 31 novembre, e valuta i range in cui giorni e mesi sono effettivamente corretti. In base a cio' decrementa e modifica gli opportuni campi.

Precondizione
la data in input deve essere corretta e i campi devono essere pieni, altriemnti potrebbero verificarsi errori logoci a runtime
Postcondizione
la data restituita deve essere esattamente quella del giorno precedente all'input
