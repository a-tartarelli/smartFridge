float  kcal_da_cercare,

)

Funzione per effettuare la ricerca di una o piu' ricette in base a una soglia di kcal passata come parametro. Se le kcal della ricetta sono <= del parametro "kcal_da_cercare" allora la ricetta sara' visualizzata in formato tabellare

Parametri
ricettariovettore di tipo ricetta
numero_ricettelunghezza del vettore contenente le ricette
kcal_da_cercaresoglia di kcal
is_ordinatoe' un valore sentinella che indica se il vettore di ricette e' ordinato in modo crescente in base alla kcal (is_ordinato = 1)
Restituisce
1 se c'e' almeno una ricetta con kcal <= del parametro "kcal_da_cercare"
0 se ci sono ricette idonee
