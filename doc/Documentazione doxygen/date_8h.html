Descrizione dettagliata

Questo file contiene tutte le funzioni necessarie per gestire le date in un programma, come ad esempio l'acquisizione della data corrente dal sistema, la modifica di una data ecc. Pe rla maggior parte tali funzioni lavorano sui campi della struct data: giorno, mese, anno e giono settimanale.

◆ confronta_date()

unsigned short int confronta_date ( data  d1,
data  d2 
)

Questa funzione prende in input due parametri di tipo 'data' e se d1 < d2 restituisce 0 se d1 > d2 restituisce 1 se d1 = d2 restituisce 2

es. 3/6/2018, 5/6/2018 –> 0 1/7/2018, 30/6/2018 –> 1 1/7/2018, 1/7/2018 –> 2

Parametri
d1Variabile di tipo 'data' con i propri campi opportunamente riempiti
d2Variabile di tipo 'data' con i propri campi opportunamente riempiti
Restituisce
unsigned short int: Intero compreso tra 0 e 2

La funzione confronta prima l'anno, poi il mese e poi il giorno. Se tutti i contronti danno come risultato dei valori uguali, le due date saranno uguali. Se invece una data e' minore dell'altra lo si evince dal confronto in sequenza di anno, mese e giorno. Lo stesso vale per verificare che una data e' maggiore dell'altra.

Precondizione
le date in input devono essere corrette e i campi devono essere pieni, altriemnti potrebbero verificarsi errori logoci a runtime
Postcondizione
l'intero restituito deve essere coerente con il confronto delle date (es. non deve restituire 0 se d1 > d2)

◆ giorno_precedente()

data giorno_precedente ( data  data_corrente)

Questa funzione prende in input un parametro di tipo 'data', la modifica, decrementandola di un giorno e la data modificata viene restituita come valore di ritorno. es. 28/10/2018 –> 27/10/2018 1/1/2019 –> 31/12/2018
